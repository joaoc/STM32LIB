Detailed Description

A STM32F0xx library to ease the programming.

Register declaration for access.

For now it only supports the STM32F030R8

Author
Joao Cabral

Based on the work of:
https://github.com/kensmith/cortex-from-scratch
https://yogiken.files.wordpress.com/2010/02/c-register-access.pdf
