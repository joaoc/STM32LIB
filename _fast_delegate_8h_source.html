533 // There are two possible implementations: the Safe method and the Evil method.
534 // DelegateMemento - Safe version
535 //
536 // This implementation is standard-compliant, but a bit tricky.
537 // A static function pointer is stored inside the class.
538 // Here are the valid values:
539 // +-- Static pointer --+--pThis --+-- pMemFunc-+-- Meaning------+
540 // | 0 | 0 | 0 | Empty |
541 // | !=0 |(dontcare)| Invoker | Static function|
542 // | 0 | !=0 | !=0* | Method call |
543 // +--------------------+----------+------------+----------------+
544 // * For Metrowerks, this can be 0. (first virtual function in a
545 // single_inheritance class).
546 // When stored stored inside a specific delegate, the 'dontcare' entries are replaced
547 // with a reference to the delegate itself. This complicates the = and == operators
548 // for the delegate class.
549 
550 // DelegateMemento - Evil version
551 //
552 // For compilers where data pointers are at least as big as code pointers, it is
553 // possible to store the function pointer in the this pointer, using another
554 // horrible_cast. In this case the DelegateMemento implementation is simple:
555 // +--pThis --+-- pMemFunc-+-- Meaning---------------------+
556 // | 0 | 0 | Empty |
557 // | !=0 | !=0* | Static function or method call|
558 // +----------+------------+-------------------------------+
559 // * For Metrowerks, this can be 0. (first virtual function in a
560 // single_inheritance class).
561 // Note that the Sun C++ and MSVC documentation explicitly state that they
562 // support static_cast between void * and function pointers.
563 
564 class DelegateMemento {
565 protected:
